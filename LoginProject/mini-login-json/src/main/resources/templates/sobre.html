O que é a Aisha?

Aisha é um assistente virtual inteligente, sarcástico e cheio de personalidade. Desenvolvido com tecnologias modernas para oferecer conversas envolventes e respostas precisas.

🤖

Inteligência Artificial

Aisha utiliza modelos de linguagem de última geração (LLM) via API da OpenAI para gerar respostas contextuais, precisas e com personalidade única.

🧠

Memória de Conversas

O sistema sumariza automaticamente o histórico de conversas e persiste no banco de dados MySQL, permitindo que a Aisha "lembre" de interações anteriores.

🔐

Autenticação Segura

Sistema completo de login e registro com sessões gerenciadas pelo Spring Boot, garantindo que apenas usuários autenticados acessem o chat.

⚡

Arquitetura Moderna

Backend em Java com Spring Boot expõe uma API REST. Um microsserviço Python com Flask integra a comunicação com a OpenAI de forma desacoplada.
